# Install dependencies

```
npm install
```

or

```
yarn install
```

# Run the app

```
npm run serve
```

# Run migrations

```
npm run migrations
```

# Undo migrations

```
npm run undomigrations
```

# Remigrate

```
npm run remigrate
```
